# Coffee Mug Detection and Tracking

## Project Overview
This project detects and tracks coffee mugs in videos using YOLOv8 and OpenCV.

## Features
- Detect coffee mugs in images and videos
- Track mugs across video frames
- Assign tracking IDs to detected mugs
- Works on custom mug videos

## Technologies Used
- Python
- YOLOv8
- OpenCV
- Ultralytics

## Files
- mug_detector.py - Mug detection
- mug_tracker.py - Mug tracking
- final_mug_tracker.py - Final tracking implementation

## How to Run

Install dependencies:

```bash
pip install ultralytics opencv-python
```

Run:

```bash
python final_mug_tracker.py
```
